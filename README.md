## Sobre o Projeto

O sistema utiliza uma **Arquitetura em Três Camadas (Cliente-Servidor)**:

1.  **Frontend:** React.js + TypeScript.
2.  **Backend:** Node.js + Express (API RESTful).
3.  **Banco de Dados:** PostgreSQL.
